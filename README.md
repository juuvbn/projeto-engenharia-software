# :handshake: Sistema de Serviços Voluntários

Uma plataforma solidária projetada para conectar **clientes** que necessitam de auxílio a **prestadores de serviço** dispostos a oferecer seu trabalho de forma gratuita e voluntária.

## :dart: Sobre o Projeto

O objetivo principal desta aplicação é facilitar o acesso a serviços para pessoas sem condições financeiras.

* **Clientes** podem utilizar a plataforma para buscar e filtrar prestadores adequados às suas necessidades.
* **Prestadores**, por sua vez, podem cadastrar suas formações técnicas, visualizar necessidades publicadas e aceitar propostas para realizar os serviços.
